## 19. Определите функцию (ЛУКОВИЦА n), строящую N-уровневый вложенный список, элементом которого на самом глубоком уровне является N ##

``` lisp
(defun luc (n &optional (s n) ) 
    (cond 
        ((eq n 0) s)
        (t(luc (- n 1) (list s)))
    ) 
) 

(print (luc 1))
(1) 

(print (luc 5))
(((((5))))) 

(print (luc 0))
0
```
